Pantalla de registro de participante:

Si accedió en la parte de Regístrate le aparecerá  una interfaz donde le permitirá llenar los datos, y completar la información correctamente.

Botón nuevo: Permite generar un Número o código del mismo para poder registrarse.

Botón Inscribir: Permite guardar los datos para el evento, esto ocurre siempre y cuando estén llenos los datos sino le presentará un error.  

Botón Listar Inscritos: Permite mostrar la tabla o matriz de los participantes registrados donde si está registrado puede iniciar sesión.  

Botón Cancelar: Permite salir de la aplicación en caso de no querer realizar ninguna acción.

Botón Cerrar Sesión: Permite salir de la aplicación en caso de no querer realizar ninguna acción.

Pantalla de inscripción al evento:

Al momento de dirigirse al menú de navegación y presionar en inicio se desplazará la opción de información de evento y mostrara la pantalla de información de evento:

En la pestaña de Registrar: permite ir a registrar un participante en el mismo se encuentra las siguientes acciones de botones:

Botón nuevo: Permite generar un Número o código del mismo para poder registrarse.

Botón Buscar: permite buscar un participante, ingresado primero su número de cedula, y se cargara los datos en los campos de textos, y permitirá actualizar o eliminar si fuera el caso, caso contrario le presentara error.  

Botón Insertar: Permite guardar los datos para el evento, esto ocurre siempre y cuando estén llenos los datos sino le presentará un error.  

Botón Actualizar: Permite actualizar los datos para el evento, esto ocurre siempre y cuando estén llenos los datos sino le presentará un error, y también realizar la acción del botón buscar.  

Botón Eliminar: Permite eliminar los datos del participante, esto ocurre siempre y cuando estén llenos los datos sino le presentará un error, y también realizar la acción del botón buscar.  

Botón Listar: Permite mostrar la tabla o matriz de los participantes registrados.

Botón Cancelar: Permite salir de la aplicación en caso de no querer realizar ninguna acción.

Pantalla de crear cuentas de usuarios:

Al dirigirse a la pestaña de la barra de  menús opción cuentas de usuarios se desplazará la opción de crear  cuentas de usuarios.

Botón nuevo: Permite generar un  código  mismo para poder registrarse.

Botón Buscar: permite buscar un usuario, ingresado primero su login, y se cargara los datos en los campos de textos, y permitirá actualizar o eliminar si fuera el caso, caso contrario le presentara error.  

Botón Insertar: Permite guardar los datos de la cuenta de usuario, esto ocurre siempre y cuando estén llenos los datos sino le presentará un error.  

Botón Actualizar: Permite actualizar los datos para la cuenta de usuario, esto ocurre siempre y cuando estén llenos los datos sino le presentará un error, y también realizar la acción del botón buscar.  

Botón Eliminar: Permite eliminar los datos de la cuenta de usuario, esto ocurre siempre y cuando estén llenos los datos sino le presentará un error, y también realizar la acción del botón buscar.  

Botón Listar: Permite mostrar la tabla o matriz de los usuarios registrados.

Botón Cancelar: Permite salir de la aplicación en caso de no querer realizar ninguna acción.
